Hexo博客访问优化 - jsDelivr 加速

CDN加速

CDN加速 是用来提高网页加载资源的速度。

hexo 部署在 github上，毕竟是国外的服务器。国内访问有点慢，而且 由于国内的CDN 加速都需要备案，所以 对于我这样的人来说。还是免费的香😁。

jsDelivr

jsDelivr 是国外的一家优秀的公共 CDN 服务提供商，也是首个「打通中国大陆（网宿公司运营）与海外的免费 CDN 服务。
最主要的是 它提供了 Github 加速服务。

Hexo 设置

在 /themes\next/_config.yml 下 找到 vendors,
可以看到 是有 jsdelivr加速的， 只是没启用。把注释去掉，并删除原来的即可
